# 🚀 Employee Management System (EMS)

A modern Full Stack Employee Management System built using **Java Spring Boot**, **React.js**, **MySQL**, and **JWT Authentication**. This application enables organizations to efficiently manage employees, attendance, payroll statistics, and departments through a responsive web interface.

---

## 📌 Features

### 🔐 Authentication

* JWT Authentication
* Secure Login & Logout
* Role-Based Access Control (Admin/Manager/User)

### 👨‍💼 Employee Management

* Add Employee
* Update Employee
* Delete Employee
* View Employee Details
* Search Employees
* Department Filter

### 📅 Attendance Management

* Mark Present / Absent / Leave
* Attendance Dashboard
* Search Attendance
* Filter by Department
* Filter by Date

### 📊 Dashboard

* Total Employees
* Present Today
* Absent Today
* Employees on Leave
* Payroll Statistics
* Department Overview Charts
* Recent Employees

### 💰 Payroll

* Total Payroll Calculation
* Salary Management

### 📱 UI

* Responsive Design
* Modern Dashboard
* Professional Login Screen
* Interactive Cards
* Attractive Charts
